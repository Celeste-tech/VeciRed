# VeciRed

Plataforma web que conecta prestadores de servicios técnicos con habitantes de las comunas de Medellín.

## Stack Tecnológico

- **Frontend:** React + Vite (Vercel)
- **Backend:** Node.js + Express (Render)
- **Base de datos:** PostgreSQL (Render)
- **Autenticación:** JWT + bcryptjs
- **Imágenes:** Cloudinary

## Estructura del proyecto

VeciRed/
├── frontend/    # React + Vite
└── backend/     # Node.js + Express
